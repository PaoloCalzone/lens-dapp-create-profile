# Lens do it
1. fork the repository

2. in your terminal:
```
mkdir my-new-project
git clone <my-forked-repository>
cp -r my-forked-repository/* /my-new-project
cd my-new-project
git init
git remote add origin  <my-github-new-repository>
git add .
git commit -m "Init"
git branch -M main
git push -u origin main
```
